# Quick Registration Website

A simple, one-click registration form for US users. Users can register instantly via email or WhatsApp.

## Features

- One-click registration via email
- One-click registration via WhatsApp
- Fully responsive design
- Cookie consent compliance
- Privacy policy and terms of service pages
- Optimized for US users
- No server-side processing required

## Deployment on GitHub Pages

1. Create a new GitHub repository named `quick-registration`
2. Upload all files from this folder to the repository

## Customization

To customize for your needs:

   - Change email address (line with `seamaylee@outlook.com`)
   - Change WhatsApp number (line with `8618774446929`)
   - Update button text and descriptions
   - Modify colors in the CSS section

2. Edit `privacy.html` and `terms.html`:
   - Update contact information
   - Modify content as needed for your specific service
